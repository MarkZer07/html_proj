APOD - O que é isso?




O Astronomy Picture Of the Day (APOD, em português: Imagem Astronômica do Dia) é uma base de dados com fotos e vídeos mantido pela NASA e pela Universidade Tecnológica de Michigan. De acordo com o website oficial, "cada dia uma imagem diferente do universo é mostrada, com uma curta explicação escrita por um astrônomo profissional".

Entrada de Dados




Pressione o botão  Buscar  após entrar com a data inicial e a data final no padrão solicitado.
Se os dois campos estiverem vazios será mostrado o APOD da data atual. Se entrar só com a data inicial será listado os APOD desta data até a data atual. Se entrar só com a data final, será mostrado somente o APOD desta data.

APOD Encontradas




Tela que mostra uma lista de APOD em função das datas de entrada, com as imagens e seus respectivos títulos e datas destribuidos em quadros envolvidos por uma moldura laranja. Caso seja um vídeo, o quadro terá uma moldura azul envolvendo a data de publicação e o título do APOD.

Página Detalhe




Tela onde é explicado, de forma resumida, o fenômeno astronômico mostrado no APOD. Na parte inferior da tela um botão flutuante permite a tradução do texto descritivo usando o Google Translator. Sendo um vídeo, temos mais um botão flutuante que faz a reprodução via YouTube.
